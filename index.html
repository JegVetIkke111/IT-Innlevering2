Hva er nettsider? 📄

Nettsider er dokumenter som befinner seg på den såkalte verdensveven, eller bedre kjent som “World Wide Web (WWW)” på engelsk. Dette fungerer ved at klienter, slik som datamaskiner og telefoner, har programvare ved navnet nettlesere, som Chrome og Firefox. Nettleserene tilgjengeliggjør det webserverene kan levere av innhold via nemlig nettsider og er nødvendig for å kunne lese de. Hver nettside har også egne og unike nettadresser, altså URL-er, som spesifiserer hvor de befinnes. På verdensveven finnes det omtrent 1,7 milliarder ulike nettsider med sine helt ulike utseender og funksjoner. Nettsider kan være for eksempel inneholde videoer, tekst, lyd, bilder animasjoner og lenker til andre nettsider. Det er også vanlig å kalle hovedsiden eller startsiden på nettsiden, hjemmesiden, som du befinner deg på nå. Samtidig er det normalt å ha flere nettsider koblet opp med hverandre ved hjelp av lenker, som gjør at man kan fordele innholdet bedre. Dette finner du også på denne nettsiden.
